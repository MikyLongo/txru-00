# TXRU-00
A level-based stealth game developed in 2025 as an individual university project for the **Digital Game Development** course at the **University of Catania**. Awarded the highest grade (**30/30 with honors**).

## 🎮 Description
TXRU-00 is a time-based level stealth game where the player must complete objectives and reach the goal while avoiding detection.
The final level features a boss battle.

**Supported languages:** Italian, English

## 🛠 Technologies
- Unity Engine (version 2022.3.36f1)
- C#
